### How To Run

#### Bash
`chmod +x OwO.sh && ./OwO.sh`

#### Batch
`CALL OwO.bat`

#### C
`gcc OwO.c -o OwO && ./OwO`

#### C++
`g++ OwO.cpp -o OwO && ./OwO`

#### C#/.NET
`dotnet restore && dotnet run`

#### Dart
`dart OwO.dart`

#### Emojicode
`emojicodec OwO.emojic && ./OwO`

#### Go
`go run OwO.go`

#### Haskell
`runghc OwO.hs`

#### HTML
`firefox OwO.html`

#### Java
`java OwO.java`

#### JavaScript (NodeJS)
`node OwO.js`

#### JScript
`cscript OwO.js`

#### Kotlin
`kotlinc OwO.kt -include-runtime -d OwO.jar && java -jar OwO.jar`

#### LaTeX
`pdflatex OwO.tex` or `latex OwO.tex` or `xelatex OwO.tex`

#### Lisp
`clisp OwO.lisp`

#### Lua
`lua OwO.lua`

#### Nim
`nim compile --run OwO.nim` or `nimble run`

#### Perl
`perl OwO.pl`

#### PHP
`php OwO.php`

#### PowerShell
`powershell OwO.ps1`

#### Python (v2.x and v3.x respectively)
`python main.py` or `python3 main.py`

#### Ruby
`ruby OwO.rb`

#### Rust
`cargo run`

#### Swift
`swift OwO.swift`

#### TCL
`tclsh OwO.tcl`
