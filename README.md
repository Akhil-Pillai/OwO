### How To Run

#### BASIC
`fbc OwO.bas`

#### Bash
`chmod +x OwO.sh && ./OwO.sh`

#### Batch
`CALL OwO.bat`

#### C
`gcc OwO.c -o OwO && ./OwO`

#### C++
`g++ OwO.cpp -o OwO && ./OwO`

#### C#/.NET
`dotnet restore && dotnet run`

#### Crystal
`crystal OwO.cr`

#### D
`dmd OwO.d && ./OwO`

#### Dart
`dart OwO.dart`

#### Elisp
`emacs -q --script OwO.el`

#### Elixir
`elixir Elixir.exs`

#### Emojicode
`emojicodec OwO.emojic && ./OwO`

#### Fetlang
`fetlang OwO.fet`

#### Go
`go run OwO.go`

#### Haskell
`runghc OwO.hs`

#### HTML
`firefox OwO.html`

#### Java
`java OwO.java`

#### JavaScript (NodeJS)
`node OwO.js`

#### JScript
`cscript OwO.js`

#### Kotlin
`kotlinc OwO.kt -include-runtime -d OwO.jar && java -jar OwO.jar`

#### LaTeX
`pdflatex OwO.tex` or `latex OwO.tex` or `xelatex OwO.tex`
